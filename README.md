## Installation

Clone from github

```bash
  git clone https://github.com/naemazam/Remember-Password.git
```
Run it

```python
  python3 R-Pass.py
```
    
  
  # User Guide
  For save Data: 
  - Put Your website name  
  - Mail address or  UserId 
  - Put Password or Genarate It. 
  - Click on Save to Save it
  ![](./rpass2.PNG)
  
  For Find Data: 
  - Put Your website name
  - Click *Search*.
